From Rapid Api

This API is used to get data about NBA teams and it's players.

The options I used to get the team names was the /teams endpoint. I added a page=0 option to only get results of the first page, since I am only showing three teams.

This API is useful if you want to create an application where users can see player or teams data, and track their performance.

It can also be useful if you want to create a Basketball game. You can get two team names and its players for matches.
